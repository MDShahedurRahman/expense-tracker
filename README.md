# Expense Tracker

This is a simple Python command-line expense tracker.

The application allows users to:
- Add expenses
- Store expenses in a JSON file
- Calculate total expenses

This project is built using basic Python concepts and file handling.

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run the application:

```bash
python main.py
